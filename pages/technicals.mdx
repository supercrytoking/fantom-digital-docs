# Technicals

Fantom Digital is 100% open source.

Fantom Digital is built primarly in TypeScript & JavaScript, utilizing the
latest techologies avaiable for the web.

Most of the hard work is done on the front-end, but the Fantom Digital also has a micro-service back-end running in Node to verify incoming payments and carry out NFT creation.

The contract, written in Solidity, leverages [OpenZepplin](/) to minimize risk by using battle-tested libraries of smart contracts for Ethereum and other blockchains.
